## 비트

- 정보를 저장하고 연산을 수행하기 위해 컴퓨터는 비트(bit)라는 측정 단위를 씀
- 비트는 이진 숫자라는 뜻을 가진 'binary digit'의 줄임말임
- 0과 1, 두 가지 값만 가질 수 있는 측정 단위

## 비트열

- 여러 숫자 조합을 컴퓨터에 나타내기 위해 비트열을 사용함
- 바이트(byte)는 여덟 개의 비트가 모여 만들어진 것
